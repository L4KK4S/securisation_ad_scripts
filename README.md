# securisation_ad_scripts

Ce repository s'inscrit dans le projet : Audit et sécurisation d’un domaine AD vulnérable dans le cadre du module ARAS83-2526PSP01 - Administration et sécurité Windows (INGE-2-APP-RS)
Ces scripts s'appuient sur les recommandations de l'ANSSI concernant la sécurisation d'un AD disponible [ici](https://www.cert.ssi.gouv.fr/uploads/ad_checklist.html)

Les deux points choisis sont :
* Zones DNS mal configurées (ici)
* Comptes utilisateurs de carte à puce sans expiration de mot de passe (ici)
